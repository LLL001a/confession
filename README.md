
# 表白网站confession
### 一个纯静态的表白/求婚网站，5个页面循序渐进，用文字、视频与照片墙传递心意。
**示例页面**：https://lll001a.github.io/confession/

## 项目结构
```text
confess/
├── index.html      # 第1页：开场小游戏
├── 2/
│   └── index.html  # 第2页：天数计时器
├── movie/
│   └── index.html  # 第3页：视频播放页
├── 4/
│   ├── index.html  # 第4页：时光放映机（旋转照片墙）
│   └── photos/     # 照片存放目录
├── 5/
│   └── index.html  # 第5页：收尾/表白页
└── README.md
```
---

## 快速开始
1.  **下载源码**：克隆或下载本项目到本地。
2.  **部署**：将整个文件夹上传到 Web 服务器（Nginx/Apache/静态托管平台），或直接在本地用浏览器打开 `index.html` 预览。
3.  **访问**：通过服务器地址访问根目录 `index.html`，即可浏览完整表白流程。

---

## 页面使用说明
### 🎬 第2页：天数计时器
- 修改代码中的日期为你需要的日期即可：
```text
 // ========== 时间计算 ==========
  const startDate = new Date('2021-12-18 10:30:00');
```
### 🎬 第3页：视频播放页（`movie` 文件夹）
- 准备你要播放的视频。
- 将视频重命名为 `movie.mp4`。
- 放入 `movie` 文件夹，覆盖原有文件。
- 刷新页面即可自动播放。

### 🖼️ 第4页：时光放映机（`4` 文件夹）
- 准备 **10 张照片**（支持 JPG/PNG）。
- 按顺序重命名为 `1.jpg`、`2.jpg`…`10.jpg`。
- 全部放入 `4/photos` 文件夹，覆盖原有文件。
- 进入第4页即可看到旋转照片墙效果。

### ✍️ 文本与样式自定义
- 打开任意 `.html` 文件，直接修改中文文本，替换成你想对 TA 说的话。
- 如需调整颜色、字体，可编辑页面内联 CSS 代码。
- 修改后刷新浏览器即可生效。

## 小贴士
- 压缩图片、视频素材，加快页面加载速度。
- 部署到 GitHub Pages：将仓库推送到 `gh-pages` 分支即可在线访问。
- 本项目为纯前端静态页面，**无数据收集功能**，所有内容仅在浏览器端运行。
- 更换背景音乐：替换代码中的链接即可：
```text
<audio id="bgMusic" loop preload="auto">
    <source src="https://music.163.com/song/media/outer/url?id=1899437032.mp3" type="audio/mpeg">
</audio>
```
---

## 注意事项
- 请确保你拥有所使用图片、视频等素材的版权或授权。
- 视频文件名必须为 `movie.mp4`，照片必须命名为1-10，如：'1.jpg、2.jpg...10.jpg'否则页面无法正常加载。
